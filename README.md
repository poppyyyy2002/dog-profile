# Dog Profile Pro 🐾

<img src="dog profile photo.png" alt="Dog Profile Pro Cover" width="100%">

## Description ✨ 
A sleek, interactive Dog Profile tool and Mood Quiz. This project provides personalized pet care advice and fun mood predictions with a responsive, user-friendly interface.

一款精緻的狗狗檔案與心情測驗工具。本專案透過直覺的互動介面，提供個人化的寵物照護建議與趣味心情預測。

## Key Features ✨ 

**Dynamic Advice:** Tailored care suggestions based on breed, age, and activity.
 動態照顧建議：根據品種、年齡與活動量提供建議。

**Doggy Mood Quiz:** A fun interactive section to check your pup's mood.
 小小測驗：趣味互動功能，一鍵查看狗狗心情。

**Adaptive UI:** Responsive speech bubbles and multilingual support (EN/ZH).
 適應性介面：自動縮放氣泡框與中英雙語切換。
